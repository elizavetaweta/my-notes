# 📜 Аттестация объектов информатизации

> [!abstract] Определение
> **Аттестация** — это комплексная проверка объекта (информационной системы, ЛВС, защищаемого помещения), в ходе которой подтверждается его соответствие требованиям безопасности.

### 🎯 Что проверяется при аттестации:
Проверяется не только компьютер, а вся среда:
1. **Программное обеспечение:** Наличие сертифицированных [[Средства защиты информации (СЗИ)|СЗИ]] (антивирусы, ОС).
2. **Техническое оборудование:** Правильность монтажа, экранирование.
3. **Физическая среда:** Замки на дверях, решетки на окнах, отсутствие «жучков».
4. **Документация:** Наличие приказов, журналов учета, должностных инструкций.
5. **Персонал:** Знают ли сотрудники правила работы с [[1.2_Информация с ограниченным доступом (тайны)]].

### 👥 Кто проводит:
Только **Органы по аттестации** — компании, имеющие специальную [[Лицензирование деятельности по ТЗИ|лицензию ФСТЭК России]]. 
*Сам себя аттестовать ты не можешь (принцип независимости).*

### 📈 Этапы аттестации (Чек-лист):
- [ ] **Заявка:** Подача обращения в орган по аттестации.
- [ ] **Предварительный анализ:** Изучение документации и структуры системы.
- [ ] **Испытания:** Технические замеры (поиск [[Защита от утечек по техническим каналам|каналов утечки]]), попытки взлома.
- [ ] **Заключение:** Отчет о выявленных недостатках и их устранении.
- [ ] **Выдача Аттестата:** Документ, разрешающий обработку секретной информации (обычно на срок 3–5 лет).

---

### ⚖️ Когда аттестация ОБЯЗАТЕЛЬНА:
1. При работе с **Государственной тайной**.
2. Для **[[ГИС (Государственные информационные системы)|ГИС]]** (согласно [[Приказ ФСТЭК №17]]).
3. Для значимых объектов **[[КИИ (Критическая информационная инфраструктура)|КИИ]]**.
4. В остальных случаях (ПДн, Коммерческая тайна) аттестация может быть добровольной.

> [!warning] Важно
> Если в систему внесены серьезные изменения (добавили новый сервер, переехали в другой кабинет), аттестация должна быть проведена заново.
