#regulator #tech


> [!abstract] Что это такое?
> Это шкала надежности средств защиты информации ([[СЗИ]]). Чем ниже цифра уровня, тем выше требования к проверке софта и тем более секретные данные им можно защищать.

### 📊 Классификация уровней (всего 6):


| Уровень       | Где применяется?      | Что защищаем?                      |     |
| :------------ | :-------------------- | :--------------------------------- | --- |
| **1 уровень** | Государственная тайна | Сведения «Особой важности»         |     |
| **2 уровень** | Государственная тайна | Сведения «Совершенно секретно»     |     |
| **3 уровень** | Государственная тайна | Сведения «Секретно»                |     |
| **4 уровень** | [[Приказ ФСТЭК №17]]  | Значимые объекты, К1, УЗ-1         |     |
| **5 уровень** | [[Приказ ФСТЭК №21]]  | Системы среднего уровня (УЗ-2)     |     |
| **6 уровень** | [[Приказ ФСТЭК №21]]  | Самые простые системы (УЗ-3, УЗ-4) |     |

---

### 🔍 Что проверяют на разных уровнях:
- **6 и 5 уровни:** Тестирование функций безопасности (работает ли антивирус так, как заявлено).
- **4 уровень и выше:** Обязательный **анализ исходного кода** на наличие уязвимостей и недекларированных возможностей («закладок»).

### 🔗 Связь с практикой
Если я как [[Специалист по ИБ]] закупаю межсетевой экран для системы класса **К1** (по 17-му приказу), я обязана выбрать устройство, имеющее сертификат не ниже **4 уровня доверия**.
