#law 


> [!abstract] Суть
> Государственная политика по принудительному переходу с иностранного ПО и оборудования (Cisco, Microsoft, Oracle) на российские аналоги.

### 📜 Ключевые указы:
- **Указ Президента №166:** Запрет на закупку иностранного ПО для значимых объектов [[КИИ]].
- **Указ Президента №250:** Усиление полномочий отделов ИБ. Запрет на использование СЗИ (средств защиты) из «недружественных» стран с 2025 года.

### 🛠 Что это значит для специалиста:
- Нужно уметь работать с **Astra Linux**, **Alt Linux** вместо Windows.
- Нужно знать российские [[СЗИ]]: **ViPNet**, **UserGate**, **Positive Technologies**, **Kaspersky**.
- Обязательная проверка ПО на отсутствие «удаленного управления» из-за рубежа.
