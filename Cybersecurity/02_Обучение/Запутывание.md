# Запутывание (Confusion)

**Запутывание** — это принцип при котором усложненяет зависимость между **ключом** и **шифртекстом**. В идеале, изменение одного бита ключа должно приводить к совершенно непредсказуемому изменению более чем половины бит шифртекста. Это делает невозможным подбор ключа по частям.

### Суть
Злоумышленник не должен иметь возможности вычислить ключ, даже если он знает результаты шифрования. Если изменить один бит ключа, весь шифротекст должен измениться до неузнаваемости.

### Реализация
В современных блочных шифрах (как AES) за это отвечают **S-блоки (Substitution-boxes)** — таблицы замен, которые нелинейно преобразуют входные данные.
