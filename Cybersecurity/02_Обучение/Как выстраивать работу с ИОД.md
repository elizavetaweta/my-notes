# Алгоритм работы с информацией ограниченного доступа

Чтобы информация имела юридическую защиту, нужно пройти 3 этапа:

### 1. Юридический этап (Бумаги)
- [ ] Издать приказ об установлении режима защиты.
- [ ] Утвердить перечень сведений (что именно секретно).
- [ ] Ознакомить сотрудников под подпись.

### 2. Организационный этап (Люди)
- [ ] Нанести маркировку (грифы) на документы/носители.
- [ ] Ограничить физический доступ (сейфы, пропуски).

### 3. Технический этап (IT)
- [ ] Настроить [[Управление доступом (IAM)]].
- [ ] Использовать [[Криптография|шифрование]] и [[Пароль и парольная политика|парольная политика]]
- [ ] Внедрить системы контроля (логирование, DLP).
