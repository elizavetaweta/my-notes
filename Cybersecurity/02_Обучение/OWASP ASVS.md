# 🧪 OWASP ASVS — Application Security Verification Standard

> [!abstract] Суть одной фразой
> OWASP ASVS — это список требований к безопасности веб-приложений, который является отраслевым стандартом де-факто для разработки, аудита и пентеста. Он не будучи формальным ГОСТом, обязателен к знанию для любого специалиста по безопасности приложений.

---

## 📌 Что такое ASVS

ASVS (Application Security Verification Standard) — это фреймворк, разработанный сообществом OWASP, который определяет **уровни верификации безопасности** приложения:

- **Level 1 (Low):** базовые требования, легко проверяемые автоматически. Минимум для всех приложений.
- **Level 2 (Standard):** рекомендуется для большинства систем, обрабатывающих чувствительные данные. Требует ручного анализа и тестирования.
- **Level 3 (High):** для критичных систем (банки, медицина, госсектор). Максимальный охват требований и строгая проверка.

## 🔑 Области требований (V1–V14)

Стандарт структурирован по категориям, покрывающим все аспекты безопасности приложения:
- V1 — Архитектура и дизайн
- V2 — Аутентификация
- V3 — Управление сессиями
- V4 — Контроль доступа
- V5 — Валидация и кодирование данных
- V6 — Криптография
- V7 — Логирование и мониторинг
- V8 — Защита данных
- V9 — Безопасность коммуникаций
- V10 — Вредоносный код
- V11 — Бизнес-логика
- V12 — Файлы и ресурсы
- V13 — API и веб-сервисы
- V14 — Конфигурация

## 🔗 Связь с другими стандартами

- **ГОСТ Р 56939-2024** — ASVS может использоваться как практическая детализация требований этого ГОСТа.
- **[[OWASP Top 10]]** — «топ-10» угроз, от которых ASVS помогает защититься.
- **ISO 27001** — ASVS закрывает технические аспекты, которые входят в меры контроля Annex A (например, A.14 — безопасная разработка).

## 🧠 Значимость для меня как специалиста по ИБ

- Я могу использовать ASVS для формализации требований к разработчикам: «наше приложение должно соответствовать ASVS Level 2».
- При пентесте и аудите я проверяю соответствие приложения пунктам ASVS — это даёт объективную и признанную в индустрии оценку.
- Знание ASVS обязательно на собеседованиях на позиции Application Security и высоко ценится в вакансиях пентестеров.

## ❓ Самопроверка

1. **Сколько уровней верификации в ASVS?**  
   👉 Три: Level 1 (базовый), Level 2 (стандартный), Level 3 (высокий).
2. **Для каких систем рекомендуется Level 3?**  
   👉 Для критичных (банки, медицина, госсектор).
3. **Как ASVS связан с OWASP Top 10?**  
   👉 ASVS покрывает гораздо больше требований, но его выполнение автоматически закрывает большинство пунктов OWASP Top 10.
