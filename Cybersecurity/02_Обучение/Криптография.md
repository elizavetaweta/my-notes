#term/concept #must-know

> [!abstract] Суть
> Криптография — это наука о методах обратимого преобразования информации с помощью математических алгоритмов для сокрытия её смысла от незаконных пользователей.

Шифрование превращает понятный текст в набор символов. Чтобы его прочитать, нужен [[Ключ]]. Без ключа хакер увидит только бессмысленный набор знаков.
Криптография обеспечивает конфиденциальность, целостность и [[Подлинность|подлинность]].

## 🔑 Классификация шифров


| Тип                                                | Описание                                                                                  | Примеры                                                            |
| :------------------------------------------------- | :---------------------------------------------------------------------------------------- | :----------------------------------------------------------------- |
| **[[Симметричные криптосистемы\|Симметричные]]**   | Используют один и тот же ключ для зашифрования и расшифрования.                           | [[AES]], [[DES]]                                                   |
| **[[Асимметричные криптосистемы\|Асимметричные]]** | Используют пару ключей: открытый (для зашифрования) и закрытый (для расшифрования).       | [[RSA]], [[ECC]]                                                   |
| **Потоковые**                                      | Шифруют данные по одному биту или байту за раз.                                           | [[RC4]], [[Salsa20]]                                               |
| **Блочные**                                        | Разбивают текст на блоки фиксированного размера (например, 128 бит) и шифруют их целиком. | AES, [[Кузнечик]]                                                  |
| Подстановочные                                     | Заменяют одни символы другими по определенному правилу.                                   | [[Шифр Цезаря]], [[Атбаш]], [[шифр «Энигмы»]]                      |
| Перестановочные                                    | Меняют порядок символов в тексте, не изменяя сами символы.                                | [[«Скитала»]], [[«Квадрат Полибия»]], [[Железнодорожная изгородь]] |
| Композиционные                                     | Сочетают несколько методов (подстановку, перестановку и др.) для повышения стойкости.     | **AES**, **DES**, [[ГОСТ 28147-89]]                                |
|                                                    |                                                                                           |                                                                    |
