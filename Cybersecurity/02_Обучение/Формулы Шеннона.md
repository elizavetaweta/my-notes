# Математическая теория секретности (Клод Шеннон)

Клод Шеннон заложил математический фундамент того, что мы называем «надежным шифром».

### 1. Формула энтропии (Мера неопределенности)
Энтропия ($H$) показывает количество информации (в битах), содержащееся в сообщении:
$$H(X) = -\sum_{i=1}^{n} p(x_i) \log_2 p(x_i)$$
Где $p(x_i)$ — вероятность появления конкретного символа.
*Для криптографии:* Чем выше энтропия ключа, тем сложнее его угадать.

### 2. Условие совершенной секретности
Шеннон доказал, что шифр является **абсолютно стойким** (невозможно взломать даже при бесконечной мощности), если выполняется условие:
$$H(K) \geq H(M)$$
То есть **энтропия ключа должна быть не меньше энтропии сообщения**.

### 3. Практический вывод
Это условие реализуется только в одном алгоритме — **Одноразовый блокнот (One-Time Pad)**:
1. Ключ равен по длине сообщению.
2. Ключ абсолютно случаен.
3. Ключ используется только один раз.
