# 🕳️ Уязвимость (Vulnerability)

> [!abstract] Определение
> **Уязвимость** — это слабое место или недостаток в системе, который может быть использован для реализации угрозы.

**Простыми словами:** Это «дыра в заборе». Ответ на вопрос: «Почему нас могут взломать?».
- **Примеры:** Слабый пароль, отсутствие обновлений Windows, открытая дверь в кабинет, ошибка в коде сайта.
- **Источники:** Ошибки разработчиков, халатность админов, физические недостатки **[[ОИ]]**
- **Инструмент поиска:** [[БДУ ФСТЭК]] (реестр **[[ВУЛ]]**), сканеры уязвимостей.

«Для учета и классификации программных уязвимостей в РФ используется реестр [[БДУ ФСТЭК]], где каждой из них присваивается уникальный номер [[ВУЛ]].»
