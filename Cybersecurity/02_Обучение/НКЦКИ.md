# 🇷🇺 НКЦКИ (Национальный координационный центр по компьютерным инцидентам)

> [!abstract] Что это такое?
> **НКЦКИ** — это орган, созданный **[[ФСБ России]]** для координации деятельности по обнаружению, предупреждению и ликвидации последствий компьютерных атак на территории РФ.

### 🏠 Роль в системе
НКЦКИ является **Главным центром [[ГосСОПКА]]**. Если ГосСОПКА — это сама сеть и правила, то НКЦКИ — это люди и эксперты, которые управляют этой сетью из центра.

### 📋 Основные функции для специалиста по ИБ:
1. **Прием уведомлений об инцидентах:** Сюда ты отправляешь отчет, если твою компанию взломали (обязательно для субъектов **[[ФЗ-187|КИИ]]**).
2. **Рассылка предупреждений:** Центр выпускает бюллетени о новых вирусах и уязвимостях (например, "Срочно обновите Windows, обнаружена дыра").
3. **Координация:** Помощь организациям в «тушении пожара» при крупномасштабных атаках.
4. **Международное сотрудничество:** Взаимодействие с аналогичными центрами других стран (CERT).

### ⚠️ Сроки информирования (Важно!):
Согласно приказам ФСБ, при обнаружении атаки на значимый объект КИИ я должна уведомить НКЦКИ:
- **В течение 3 часов** — при инциденте на значимом объекте.
- **В течение 24 часов** — во всех остальных случаях.
