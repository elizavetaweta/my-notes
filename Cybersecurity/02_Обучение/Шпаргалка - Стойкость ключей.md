# Шпаргалка: Стойкость ключей

Актуальные данные по длине ключей (в битах) и их применимости в ИБ:


| Длина ключа | Статус | Применение |
| :--- | :--- | :--- |
| **56 бит (DES)** | ❌ Взломано | Не использовать. Взламывается за часы. |
| **80-112 бит** | ⚠️ Устаревает | Допустимо для кратковременной защиты старых систем. |
| **128 бит (AES)** | ✅ Надежно | Стандарт для банковских систем, VPN и персональных данных. |
| **256 бит (AES)** | 🛡️ Максимально | Защита гос. тайны и «квантовая устойчивость» на будущее. |

> [!TIP] Факт
> Увеличение длины ключа на 1 бит удваивает количество попыток, необходимых для взлома методом перебора (Brute-force).
