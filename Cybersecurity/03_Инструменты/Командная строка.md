`Ctrl + Alt + T`

Создать пустой файл:
```bash
touch filename.txt
```

**Редактор `nano`** (создает файл и сразу открывает его для ввода текста):
```bash
nano filename.txt
```
Для сохранения изменений в nano — Ctrl + O и Enter, а для выхода — Ctrl + X

Перенаправление вывода > (быстрое создание файла с текстом):
```bash
echo "Привет, мир!" > filename.txt
```

Отобразить абсолютный путь к текущей папке, в которой находится пользователь
**pwd** (от англ. _print working directory_):
```bash
pwd
```

Вывести список файлов внутри текущей папки:

```bash
ls -l
```
> [!NOTE]
> Команда `ls -l` расшифровывается как ==**list long**== (вывести список в длинном формате).
>  В отличие от обычной команды `ls`, которая показывает только имена файлов, флаг `-l` выводит подробную таблицу со всей информацией о каждом файле и папке.
Переместить файл (в папку внутри текущей директории):

```bash
mv [что_перемещаем] [куда_перемещаем]
mv filename.txt Documents/
```

Переместить файл из любого места (указываем полный путь):
(Символ `~` — это быстрое обозначение домашней папки).
```bash
mv [что_перемещаем] [куда_перемещаем]
mv filename.txt ~/Documents/
```
