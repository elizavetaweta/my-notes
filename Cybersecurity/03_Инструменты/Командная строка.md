`Ctrl + Alt + T`

Создать пустой файл:
```bash
touch filename.txt
```

**Редактор `nano`** (создает файл и сразу открывает его для ввода текста):
```bash
nano filename.txt
```
Для сохранения изменений в nano — Ctrl + O и Enter, а для выхода — Ctrl + X

Перенаправление вывода > (быстрое создание файла с текстом):
```bash
echo "Привет, мир!" > filename.txt
```
